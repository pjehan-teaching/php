Les tableaux

Présentation

Un tableau (array) est un type de donnée. On peut donc utiliser une variable pour manipuler des tableaux.

Index Valeur
0 Pierre
1 Eric
2 Antoine
3 Julie
4 Marie
En PHP, les tableaux commencent à l’index 0

Déclarer un tableau

Les extraits de code ci-dessous permet de créer un tableau en insérant des valeurs dans chaque cases.

<?php
// Utiliser la fonction array
$classes = array('Pierre', 'Eric', 'Antoine');
<?php
// Utiliser la syntaxe PHP7+
$classes = ['Pierre', 'Eric', 'Antoine'];
<?php
// Définir chaque index
$classes[0] = 'Pierre';
$classes[1] = 'Eric';
$classes[2] = 'Antoine';
<?php
// Définir chaque index sans préciser les numéros de case
$classes[] = 'Pierre';
$classes[] = 'Eric';
$classes[] = 'Antoine';

Les tableaux associatifs

On peut également créer des tableaux associatifs c’est-à-dire utiliser des chaînes de caractères pour remplacer les index :

<?php
// Utiliser la fonction array
$etudiant = array(
  'nom' => 'Dupont',
  'prenom' => 'Jean',
  'age' => 19
);
<?php
// Définir chaque index
$etudiant['nom'] = 'Dupont';
$etudiant['prenom'] = 'Jean';
$etudiant['age'] = 19;

Tableaux multi-dimensions

On pourra également mettre des tableaux dans des tableaux.

  1. Créez une variable $classe représentant un tableau pour lequel chacune des cases contiendra un étudiant (représenté à l’aide d’un tableau associatif).
Utilisez print_r() pour afficher votre tableau (utilisez la balise HTML <pre> pour la mise en forme)

Quelques fonctions utiles

Fonction Description
count Compte tous les éléments d'un tableau ou quelque chose d'un objet
sort Trie un tableau
in_array Indique si une valeur appartient à un tableau
array_splice Efface et remplace une portion de tableau
array_push Empile un ou plusieurs éléments à la fin d'un tableau
array_pop Dépile un élément de la fin d'un tableau
La liste complète des fonctions est disponible sur la doc en ligne
