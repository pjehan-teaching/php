Uploader des fichiers

Envoyer un fichier

Pour uploader des fichiers sur un serveur via une page web, nous pouvons utiliser un champ input de type file.

Il faut également penser à ajouter l’attribut enctype="multipart/form-data" sur la balise form pour permettre l’envoi de fichier.

<form method="POST" action="traitement.php" enctype="multipart/form-data">
  <input type="file" name="image" accept="image/*"/>
  <input type="submit" value="Envoyer"/>
</form>

Récupérer un fichier

Lors de l’envoi du formulaire, le fichier est stocké dans un dossier temporaire et les informations concernant ce fichier sont stockées dans la superglobale $_FILES.

$image_nom = $_FILES['image']['name'];
$image_type = $_FILES['image']['type'];
$image_taille = $_FILES['image']['size'];
$image_nom_temporaire = $_FILES['image']['tmp_name'];
$image_erreur = $_FILES['image']['error'];

// Déplacer le fichier temporaire vers le dossier images
move_uploaded_file($image_nom_temporaire, 'images/' . $image_nom);
